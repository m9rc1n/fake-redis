# Contributor Covenant Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Report unacceptable behavior to the maintainers via a [private security advisory](https://github.com/marcinnurbanski/fake-redis/security/advisories/new) or by email. All reports are handled confidentially.
